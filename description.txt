QoL mod which for the time being just adds scrollbars to both buffs' bar UIs in the game.
This mod should be compatible with any mod which doesn't replace vanilla UI as well.
This mod doesn't work whith gamepads.
Make sure to look into all of the keybinds and configs available.

My GitHub profile: UnknownDevices (where you can find this mod's source code).
