[ENG] Mainly QoL mod which improves various UIs in the game while keeping them 'vanilla'.
Most features of this mod are toggable and tweakable so make sure to look into all of the configs available.

Features: #(not exhaustive)
- adds scrollbars to both buffs bars
- adds configs to customize the rows, columns and alpha of each buffs bar
- changes the locking of the hotbar to now also lock the buffs bar below it
- adds a keybind to lock/unlock the hotbar
- allows interacting with the hotbar (scrolling, clicking on, etc.) regardless of an item being in use or auto-select being on
- changes the hotbar to now scale up the item pointed to by the hotbar, which won't always be the selected item
- allows scrolling to go up or down the recipes list regardless of an item being in use
- allows scrolling to zoom in or out on the fullscreen map regardless of an item being in use
- changes the inventory to now also highlight items selected by auto-select, not just manually selected ones
- fixes a vanilla bug where the hotbar is drawn for one frame when opening the bestiary or emotes window
- fixes a vanilla bug where hovered buff icons remain highlighted between toggles of the inventory

This mod most likely doesn't work whith gamepads.
Any bug report or suggestion is greatly appreciated.
