[ENG] QoL mod which enhances many in-game UIs without changing them drastically.
Most features of this mod are toggable and tweakable so make sure to have a look into the configs page.

Features: #(not exhaustive)
- adds scrollbars to both buffs bars (do note that this mod does not increase the buff slots available to a player, you can use mods such as Fargo's Mutant Mod for this)
- includes configs to customize the rows count, columns count and alpha of both buffs bar
- changes the hotbar's buffs bar to now lock when the hotbar is locked
- changes the minimap to now lock when the hotbar is locked 
- includes a keybind to lock and unlock the hotbar
- changes the hotbar to now be interactable (scrolling, clicking on, etc.) regardless of an item being in use
- changes the hotbar to now scale up the item pointed to by the hotbar, which won't always be the selected item
- changes the recipes list to now be scrollable regardless of an item being in use
- changes the fullscreen map to now be zoomable regardless of an item being in use
- changes the inventory to now highlight any selected item, not just hotbar ones
- fixes a vanilla bug where the hotbar was drawn for one frame when opening the bestiary or emotes window
- fixes a vanilla bug where hovered buff icons remained highlighted between toggles of the inventory

This mod most likely doesn't work with gamepads.
Any bug report or suggestion is greatly appreciated.
