Mainly QoL mod which tweaks vanilla UIs in mayor and minor ways, remarkably, it adds scrollbars to both buffs bars and allows interacting with the hotbar while using an item or having auto-select on.
Most features of this mod are toggable and tweakable so make sure to look into all of the keybinds and configs available.
This mod is completely client side except for a single ignorable server side config
 
This mod should be compatible with any mod which doesn't replace vanilla UIs as well.
This mod doesn't work whith gamepads.
Any bug report or suggestion is greatly appreciated.

Mod principalmente de Calidad de Vida que realiza cambios para mejor a varias UIs vanilla, notablemente, añade barras de desplazamiento a ambas barras de bufo y permite interactuar con la barra de acceso rápido mientras se usa un ítem o el auto-seleccionado esta activo.
La mayoría de las características de este mod son deshabilitables y modificables así que asegurate de tomar un vistazo a todas las asignaciones de teclas y configuraciones disponibles.
Este mod es completamente del lado del cliente excepto por una sola configuración ignorable del lado del servidor.

Este mod debería ser compatible con cualquier otro mod que no reemplace UIs vanilla también.
Este mod no funciona con mandos.
Cualquier reporte de bug o sugerencia es apreciada.
