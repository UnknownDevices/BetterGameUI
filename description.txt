Mainly QoL mod which changes for the better some vanilla UIs, remarkably, it adds scrollbars to both buffs' bar and allows interacting with the hotbar while using an item.
Make sure to look into all of the keybinds and configs available.

This mod should be compatible with any mod which doesn't replace vanilla UIs as well.
This mod doesn't work whith gamepads.
Any bug report or suggestion is greatly appreciated.
